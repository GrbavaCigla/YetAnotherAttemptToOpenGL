# CiglaGL
Yet another attempt at creating game engine/game library in OpenGL

## Usage
### Dependencies
- glfw
- glm
- spdlog
- stb

Before compiling run `scripts/glad.py`  
Next compile with cmake:
```
mkdir build
cd build
cmake ..
make -j4
```

## Goal
Create basic game engine/game library I can use in basic 3D FPS games.
